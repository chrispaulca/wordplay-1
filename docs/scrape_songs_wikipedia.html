scrape_songs_wikipedia module¶

@author: Kaya Tollas

Scrapes songs from Wikipedia. Functions clean_data, write_html_table_to_csv, and scrape_singles towards end of file are from https://github.com/rocheio/wiki-table-scrape/blob/master/wikitablescrape.py

scrape_songs_wikipedia.clean_data(row)[source]¶

Clean table row list from Wikipedia into a string for CSV.

ARGS:

row (bs4.ResultSet): The bs4 result set being cleaned for output.

RETURNS:

cleaned_cells (list[str]): List of cleaned text items in this row.
scrape_songs_wikipedia.scrape_albums_1956_2013(url)[source]¶

Returns lists of artists and urls for Billboards that year (1956-2013)

ARGS:

url (str): The URL of the Wikipedia article to scrape from.

scrape_songs_wikipedia.scrape_albums_2014_2017(url)[source]¶

Returns lists of artists and urls for Billboards that year (2014-2017)

ARGS:

url (str): The URL of the Wikipedia article to scrape from.

scrape_songs_wikipedia.scrape_billboards(start_url, search_str, singles=False)[source]¶

Returns list of Billboard links (one for each year)

ARGS:

start_url (str): The URL of the Wikipedia article to scrape tables from.

search_str (str): The title of the table with the series of years

scrape_songs_wikipedia.scrape_singles(url, folder_name=None)[source]¶

Create CSVs from all tables in a Wikipedia article.

ARGS:

url (str): The URL of the Wikipedia article to scrape tables from.

output_name (str): The base file name (without filepath) to write to.

scrape_songs_wikipedia.songs_from_album(album_url)[source]¶

Returns list of songs from Wikipedia album page

ARGS:

album_url (str): The URL of the Wikipedia page to scrape from.

scrape_songs_wikipedia.songs_from_artists_albums_year(artists, album_urls)[source]¶

Create dataframe of all songs in a list of albums in a year

ARGS:

albums_url (str): The URL of the Wikipedia article to scrape from.

scrape_songs_wikipedia.write_html_table_to_csv(table, writer)[source]¶

Write HTML table from Wikipedia to a CSV file.

ARGS:

table (bs4.Tag): The bs4 Tag object being analyzed.

writer (csv.writer): The csv Writer object creating the output.
